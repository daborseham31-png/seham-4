AWS Basics for Students

AWS (Amazon Web Services) is a cloud platform that lets you build and run applications without managing physical servers. It's like renting computer power on demand!

What is Cloud Computing?

Cloud computing means using remote servers over the internet instead of your local computer. AWS provides services for storage, computing, databases, and more.

Key AWS Services

  • EC2 (Elastic Compute Cloud): Virtual servers to run your apps.
  • S3 (Simple Storage Service): Store files and data securely.
  • IAM (Identity and Access Management): Control who can access your resources.
  • Lambda: Run code without servers (serverless).

Student Example: Launch an EC2 Instance

1. Sign up for AWS Free Tier.
2. Go to EC2 dashboard.
3. Launch a free t2.micro instance.
4. Connect via SSH and run a simple command like "echo Hello World".

Remember to stop instances to avoid charges!
