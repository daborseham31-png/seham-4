GitHub Basics for Students

GitHub is a platform for version control and collaboration using Git. It helps you track changes in your code and work with others.

What is Git?

Git is a tool to manage versions of your files. It lets you save snapshots of your work and revert if needed.

Key GitHub Features

  • Repositories: Folders for your projects.
  • Commits: Save changes with messages.
  • Branches: Work on features without affecting main code.
  • Pull Requests: Propose changes for review.
  • Issues: Track bugs and tasks.

Student Example: Create a Repository

1. Sign up for GitHub.
2. Click "New repository".
3. Name it "my-first-repo".
4. Add a README.md file.
5. Commit and push your changes.

Use Git commands like git init, git add, git commit, git push.
